# WhatsDeX Technology Stack

## 1. Frontend

- **Framework:** Next.js 14+ (App Router)
- **Library:** React 18+
- **Language:** TypeScript (Strict) - **No `.js` or `.jsx` allowed.**
- **File Extensions:** `.tsx` (Components), `.ts` (Logic/Hooks)
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Data Fetching:** Axios / TanStack Table

## 2. Backend

- **Runtime:** Node.js
- **Framework:** Express
- **Language:** TypeScript (Strict) - **No `.js` allowed.**
- **File Extensions:** `.ts`
- **WhatsApp API:** Baileys (@whiskeysockets/baileys)
- **Authentication:** Firebase Admin / JWT
- **Real-time:** Socket.io
- **Caching:** Node-Cache / Redis (planned/supported)
- **Logging:** Pino / Winston

## 3. Infrastructure & Services

- **Database:** Firebase Firestore
- **Payments:** Stripe
- **File Storage:** Firebase Storage / Google Drive (backups)
- **AI Integration:** OpenAI / Google Gemini

## 4. Development & Tooling

- **Testing:** Jest, Supertest, React Testing Library
- **Linting:** ESLint 9+, Prettier
- **Git Hooks:** Husky, Lint-staged
- **Environment:** Dotenv
- **API Validation:** Zod
